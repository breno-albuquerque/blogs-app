## Install and Run:

* Dependencies:
  - Node (16.14.0)

* Clone and Install package.json dependencies:
```
$ git clone git@github.com:breno-albuquerque/blogs-api-front-end.git
$ cd blogs-api-front-end
$ npm install
```
  
* Running the Project:
  - ```$ npm start```
